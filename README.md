# Race-Pace-Graph
![Python](https://img.shields.io/badge/Python-3.10+-blue)
![Version](https://img.shields.io/badge/Version-1.0.0-green)

I developed a Race Pace Analyzer in Python using the OpenF1 API. The project allows users to compare race pace data across all drivers, race by race, from the 2023 season through 2025.

How to Run

This project is a PyQt6-based F1 Race Pace Analyzer that fetches live race data from the OpenF1 API
 and visualizes driversâ€™ lap times using Plotly. Follow the steps below to run it locally.

1. Prerequisites

Make sure you have Python 3.10+ installed. You also need the following Python packages:

PyQt6

PyQt6-WebEngine

pandas

plotly

You can install them using pip:

pip install PyQt6 PyQt6-WebEngine pandas plotly

2. Clone this repository to your local machine
3. Run the Application
